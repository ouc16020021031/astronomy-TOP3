### 环境

env.txt：环境搭建指令

requirements.txt：所需工具包

### 程序

pre.py：预提取和预处理

resnet.py：模型函数

tools.py：程序所需的函数工具

train.py：模型训练

pred.py：模型预测

sub.py：观察验证集结果和生成提交结果

main.py：一键调用 pre.py，train.py，pred.py，sub.py 4个程序，分别执行预处理，训练，预测，结果生成四个功能。
